Жасанды интеллект

Жасанды интеллект (ЖИ) — компьютерлік жүйелердің адам зиятына тән міндеттерді орындай алу қабілеті. Бұған оқу, сөйлесу, суреттерді тану, шешім қабылдау жатады.

Заманауи ЖИ жүйелері терең оқыту (deep learning) және трансформер архитектурасы негізінде жұмыс істейді. Миллиардтаған параметрлері бар нейрондық желілер арқылы адамның тілін түсінеді.

Smart Helper жобасы Anthropic Claude API пайдаланады — бұл ең қауіпсіз және дәл жауап беретін LLM модельдерінің бірі.

// Техникалық мәліметтер
МодельClaude Sonnet 4
Параметрлер~200B+
АрхитектураTransformer
Контекст200K токен
ЖасаушыAnthropic
API нұсқасы2023-06-01
// Жоба архитектурасы

Smart Helper қалай жұмыс істейді?

01
Пайдаланушы сұрақ жазады

Браузерде жазылған мәтін JavaScript арқылы өңделеді

02
API сұранысы жіберіледі

fetch() арқылы Anthropic серверіне POST сұранысы жіберіледі, таңдалған режим system prompt ретінде қосылады

03
Claude моделі өңдейді

LLM модель контексті талдап, ең дәл жауапты генерациялайды

04
Жауап қайтарылады

JSON форматындағы жауап браузерге келіп, экранда көрсетіледі. Тіркелген пайдаланушыларда Firebase-ке сақталады

05
Пайдаланушы бағалайды

👍/👎 бағасы localStorage және Firebase-ке жазылып, статистика бетінде қорытындыланады
